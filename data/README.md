# Dataset

This project uses two datasets, which are not included in this repository due to size and licensing.

## 1. NEV Energy Management Dataset (Training)
- Source: https://www.kaggle.com/datasets/ziya07/nev-energy-management-dataset
- Used for: Training the multi-output Random Forest model
- After cleaning: 4,806 rows

## 2. BMW i3 Driving-Cycle Data (Validation)
- Source: From our professor
- Used for: Real-world validation
- 70 trips, ~789K rows
